% Feature | 93857 | Resizable navigation component for all element / record selectors

\begin{frame}[fragile]
	\frametitle{Backend User Interface}
	\framesubtitle{Resizable Navigation}

	% @TODO: update text

	The newly introduced possibility to resize and/or collapse the navigation frame
	(e.g. page tree) in the main backend has been additionally added to all Element
	Browser/Record Selectors, and Link Picker selections.

	All modal areas with a Page Tree or File-based Folder Tree now contain the same
	feature-set of collapsing/resizing.
